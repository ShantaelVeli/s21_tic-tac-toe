# Tic-Tac-Toe Backend (Go)

Backend-сервис для игры «Крестики-нолики», реализованный на языке Go.

Проект демонстрирует реализацию REST API сервиса с использованием принципов Clean Architecture.

## Стек технологий

- Go
- net/http
- REST API
- Clean Architecture

## Архитектура

Проект реализован с использованием принципов Clean Architecture.

Основные слои:

- **domain** — бизнес логика игры
- **usecase** — сценарии использования
- **controllers** — HTTP обработчики
- **adapter/inmemory** — слой хранения данных (in-memory)
- **di** — подключение зависимостей при помощи uber/fx

## Функциональность

Сервис поддерживает:

- создание новой игры
- выполнение хода
- выполнение хода при помощи алгоритма minimax
- определение победителя

## API

### Создать игру

GET /game/ - создать новую игру

POST /game/{uuid} - обновление игры

Ответ:

```json
{
  "uuid": "123",
  "field": [
    [0,0,0],
    [0,0,0],
    [0,0,0]
  ],
  "resultgame": "ongoing"
}
```

## Запуск сервиса

Используйте команду `go run cmd/app/main.go` для запуска сервиса.
